# product-ms

This micro service provide API(s) for Product view and search related funclities. The service is built in Python(version 3.11.5) using FastAPI.

### Commands to run:
 - python -m venv venv
 - python -m uvicorn main:app --app-dir ./src --host 127.0.0.1 --port 8092 --reload
 
### Swagger documentation:
http://localhost:8092/docs
